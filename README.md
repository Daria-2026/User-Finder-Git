## Описание

Приложение «GitHub User Finder» позволяет искать пользователей GitHub через официальный API, отображать их данные и сохранять избранных пользователей в локальный JSON‑файл.

## Функциональность

* Поиск пользователей GitHub по имени.
* Отображение результатов поиска в виде списка.
* Добавление пользователей в избранное.
* Сохранение избранных пользователей в JSON‑файл (`favorites.json`).
* Проверка корректности ввода (поле поиска не может быть пустым).

## Использование API GitHub

Приложение использует GitHub API (v3) для получения данных о пользователях. Базовый URL: `https://api.github.com/users/`.

API не требует аутентификации для простых запросов, но имеет ограничения по количеству запросов в час.

## Требования

* Python 3.6+
* Библиотеки: `requests`, `tkinter`

## Примеры использования
Поиск пользователя:
* Введите имя пользователя в поле ввода.
* Нажмите кнопку «Найти».
* В списке результатов появится информация о пользователе.
* Добавление в избранное:
* Выберите пользователя из списка результатов.
* Нажмите кнопку «Добавить в избранное».
* Пользователь появится в списке избранных.

Проверка сохранения:
* Закройте приложение.
* Откройте файл favorites.json — в нём будет сохранён выбранный пользователь.
* Запустите приложение снова — избранные пользователи загрузятся автоматически.

Тестирование
* Корректный ввод: Введите существующее имя пользователя. Приложение должно отобразить данные.
* Некорректный ввод: Оставьте поле пустым и нажмите «Найти» — появится сообщение об ошибке.
* Неизвестный пользователь: Введите несуществующий логин — приложение покажет ошибку «Пользователь не найден».

Добавление в избранное: Добавьте пользователя и проверьте файл favorites.json.
